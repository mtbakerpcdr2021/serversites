How to Sync FTSCoin Blockchain with Bootstrap

This guide will help you install a recent copy of the blockchain. This should significantly speed up the task of getting the blockchain synced up so you can use your wallet.


Windows:


  1. Make sure FTSCoind.exe, FTSCoin-service.exe, and/or the GUI wallet is not running.

  2. Open File Explorer and type %APPDATA%\FTSCoin and hit enter.



    3. Note: if the folder doesn't exist, just go to %APPDATA% instead and create a folder named FTSCoin.


  1. Download the latest snapshot of the blockchain.

  2. Move the two new downloaded files to the %APPDATA%\FTSCoin folder.

  3. Start FTSCoind.exe or the GUI wallet like you normally do.

  4. See Expected Results section below.
